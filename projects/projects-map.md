# Projects Map

One folder per active workstream. Each project folder contains an `Overview.md` and a `WORK.md` (for ongoing multi-session work).

## Structure

```
projects/
└── {project-name}/
    ├── Overview.md     ← goal, stack, links
    └── WORK.md         ← running work note for this project
```

> Add entries here as you start new projects.

## Starting a new project

1. Create `projects/{name}/Overview.md` using `templates/project-template.md`
2. Create `projects/{name}/WORK.md` for the first task
3. Add an entry to this map
