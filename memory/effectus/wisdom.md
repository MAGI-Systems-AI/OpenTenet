# Wisdom

> **Hard-won lessons, written down so they don't need to be re-earned.** Each entry is one imperative sentence that came out of a real ISC failure loop or postmortem — not general advice, not model training knowledge. If you cannot name the task-slug that produced it, it is opinion, not wisdom.

## Write path from LEARN

Two triggers mandate a wisdom entry:

| Trigger | What to write |
|---------|--------------|
| ISC failure loop fired 2+ times on the same assumption | The assumption that kept failing, compressed to one imperative sentence |
| A task revealed a fundamental mis-scoping of the problem | The re-scoping insight, one imperative sentence |

Format: one sentence. Imperative. Past-tense origin in parens. Provenance tag at end.

Example: `Never assume a config key exists without reading the file first — silent KeyError masked the real bug for two loops. · earned from: api-key-fix · 2026-05-30`
